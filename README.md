# Greenden 🌿

Greenden is a responsive plant store website built using HTML and Tailwind CSS.  
The project features a modern UI design with responsive navigation, product sections, and a clean layout for both desktop and mobile screens.

## ✨ Features

- Responsive Navbar
- Mobile Friendly Design
- Product Page
- Search Bar UI
- Clean and Modern Layout
- Tailwind CSS Styling

## 🛠 Technologies Used

- HTML5
- Tailwind CSS

## 📂 Pages

- Home Page
- Product Page
- Contact Page

## 🚀 Future Improvements

- Add JavaScript Search Functionality
- Shopping Cart Feature
- Dark Mode
- Backend Integration
